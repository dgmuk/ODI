Об опроснике

Индекс инвалидности Освестри (Oswestry Disability Index) — это инструмент для оценки степени ограничения жизнедеятельности у пациентов с болями в поясничном отделе позвоночника.

Инструкция: В каждом разделе выберите только один вариант ответа, который наиболее точно описывает ваше состояние на сегодняшний день. Если ни один вариант не подходит идеально, выберите наиболее близкий.
